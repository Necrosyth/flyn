- Tech Stack
    1. **Automation Engine is the core brain**
        - Deterministic execution
        - Event-driven
        - Plugin-based
    2. **Workflow builder is declarative**
        - UI builds intent
        - Compiler generates executable graph
    3. **Execution semantics are centralized**
        - wait / resume
        - approval
        - retries
        - branching
    4. **Domain logic is fully pluggable**
        - voice
        - CRM
        - AI
        - HR
    5. **AI agents are executors, not orchestrators**
    6. **Telephony is an event source, not a workflow engine**

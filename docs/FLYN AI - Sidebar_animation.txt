2️⃣ Sidebar Animation Principles (Core Rules)
Design Philosophy
● Subtle, fast, confidence-inspiring
● Never distract from conversations
● Motion communicates hierarchy & intent
Golden Rules
● All animations < 250ms
● Easing: cubic-bezier(0.4, 0, 0.2, 1)
● GPU-accelerated transforms only
● No animation chaining beyond 2 steps
3️⃣ Expand / Collapse Animation
Trigger
● Click hamburger icon
● Keyboard shortcut ⌘ + \
● Auto-collapse at viewport < 1100px
Animation Spec
Property: width
From: 72px → 264px
Duration: 180ms
Easing: ease-out

Child Element Behavior
   Element                     Behavior
Icons         Fade-in text label (80ms delay)
Text          Opacity 0 → 1
Badges        Slide-in + fade
Tooltips      Disabled in expanded mode
